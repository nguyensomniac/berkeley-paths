Moving Berkeley is an experiment by the Daily Cal born from a curiosity of when and how different public spaces on campus are used. Using data aggregated from Moves, an activity tracking smartphone app, we will build a map that examines where UC Berkeley students spend their time on campus.

This project is open-source, and you can view the code on Github.
